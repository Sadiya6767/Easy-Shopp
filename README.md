# EasyShopp

EasyShopp is an Android e-commerce application developed using Jetpack Compose and Firebase.

## Features
- User Login & Signup
- Product Listing
- Product Details Page
- Add to Cart
- Favorites
- Orders
- Profile Page

## Technology Used
- Kotlin
- Jetpack Compose
- Firebase Authentication
- Firebase Firestore

## How to Run
- Clone the repository
- Open in Android Studio
- Connect Firebase
- Run the app
